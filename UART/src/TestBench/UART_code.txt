; This is metacode for UART simulation 
;
; Syntax:
; -------
; Operation [Data]
;
; Operation:	W | N | E; 
;				W <wr_byte> - Write data
;				N <Time_in_us> - Wait for time
;               E Simulation end 
;				; Non-reported comment
;
; Examples:
; ---------
; W 4A 			send 0x4A 
; N 200			wait for 200us
;
;---------------------------------------------------------------------
